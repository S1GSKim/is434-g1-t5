-MSNBC.ipynb <br />
-NBCNews.ipynb <br />
-Telegraph.ipynb <br />
-USATODAY.ipynb <br />
-voxdotcom.ipynb <br />
-washingtonpost.ipynb <br />
-WSJ.ipynb
